We are experts in COSMOS implementation and optimization. The following are a few examples of our work with COSMOS:

  • Render data from Azure Data Lake (COSMOS) directly on HTML5 report
    • We use U-SQL based queries and COSMOS API to surface data from COSMOS on rich HTML5 visuals
    • We achieve page load time (PLT) of ~10-15 seconds through COSMOS query optimization
  • Refresh Power BI Reports on the fly with data from COSMOS
    • We use COSMOS Export API and Power BI REST API to push latest data from COSMOS to Power BI datasets
    • Set up continuous-poll or scheduled refresh
  • Custom COSMOS functionality for data manipulation
    • We create custom reducers which enhance the out of box COSMOS operators to perform complicated business logic
    • With reducers, we can use advanced C# methods to achieve data manipulations
  • Secure sensitive data on COSMOS
    • We use a custom Rijndael based encryption utility to secure PII data at rest on COSMOS or during processing on COSMOS
